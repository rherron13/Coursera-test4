Transfering code to Git Hub

  • Git Hub command
    • git add (for syncing local information with remote information).
    • git commit (for commiting information to the remote git hub)
    • git push (for final update of the remote git hub)
  • Git Hub Push Error
    • Step 1: Enter git fetch orgin
    • Step 2: Enter git merge orgin + associated branch name
